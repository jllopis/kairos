# ¿Qué es A2A?

A2A (agent-to-agent) es un protocolo para que agentes se comuniquen y colaboren
como pares. Define un modelo común de mensajes y tareas, sin exigir que los
agentes compartan su estado interno o herramientas.

## Principios

- Interoperabilidad entre frameworks y proveedores.
- Capacidades declaradas vía Agent Cards.
- Comunicación basada en tareas y mensajes.
- Integración con observabilidad y seguridad.

## Cuestion clave

A2A no es un orquestador. Es el idioma común para que agentes se coordinen,
independientemente de cómo implementen su razonamiento.
